**核心算法 - 海军法体脂率公式：**
- 男：`495 / (1.0324 - 0.19077*log10(腰-颈) + 0.15456*log10(身高)) - 450`
- 女：`495 / (1.29579 - 0.35004*log10(腰+臀-颈) + 0.22100*log10(身高)) - 450`

**体脂等级（女）：** <18% 偏瘦 / 18-28% 正常 / 28-33% 偏高 / >33% 肥胖
**体脂等级（男）：** <10% 偏瘦 / 10-20% 正常 / 20-25% 偏高 / >25% 肥胖

**REST API：**
- `POST /api/body-composition/record` - 录入/更新身体成分
- `GET /api/body-composition/today` - 今日数据
- `GET /api/body-composition/latest` - 最新数据
- `GET /api/body-composition/history?limit=30` - 历史列表
