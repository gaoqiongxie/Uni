# 轻瘦 - 内容策略规划

## 📋 策略概述

### 内容策略目标
- 建立"科学减脂"的权威品牌形象
- 驱动应用下载与用户留存
- 构建活跃的减脂社区
- 通过优质内容建立信任,提升转化

### 核心内容支柱(4大支柱)
1. **科学营养** - 营养学知识、食谱开发、饮食科普
2. **运动健康** - 运动计划、健身教程、体能进阶
3. **心理激励** - 成功故事、挑战激励、社区互动
4. **生活方式** - 日常习惯、睡眠管理、压力释放

### 2️⃣ 小红书内容

**用户定位**: 年轻女性,倾向于社区氛围,内容消费能力强

**更新频率**: 4-5条/周

**内容栏目**:
- **减脂日记** - 真实记录减脂过程 (打卡照、饮食、运动、心情)

- **运动打卡** - 家庭/办公室快速运动
- **科学知识** - 营养学、代谢、激素知识
- **素人故事** - 真实用户减脂成功记录

**内容亮点**:
- 高清摄影 (美食、身材对比照)
- 实用性强 (可直接复制执行)
- 真实分享 (非完美主义,包容性强)

**运营指标目标**:
- 点赞率: 15%+
- 收藏率: 8%+
- 转发率: 5%+

### 3️⃣ 抖音/快手内容

**用户定位**: 大众娱乐受众,内容趣味性要求高

**更新频率**: 5-7条/周

## 🎯 内容创作指南

### 品牌叙事框架

**核心故事**: 从"不相信减脂能成功" → "发现科学方法" → "坚持小改变" → "收获大蜕变"

**品牌声音**:
- **温暖**: 不强调"必须"和"禁止",而是"可以尝试"和"一起加油"
- **科学**: 所有建议都基于营养学和运动科学
- **陪伴**: 强调"我们一起",不是"你必须"
- **鼓励**: 庆祝每个小进步,包容每个挫折

### 内容创作原则

1. **真实性优先** - 展示真实用户、真实过程、真实结果
2. **科学性保证** - 所有营养建议来自营养师、医学专家审核
3. **易执行性** - 所有建议都必须是用户能实现的
4. **多样性** - 呈现不同年龄、体型、生活方式的用户
5. **持续性** - 长期陪伴而非短期打鸡血

### 禁忌内容

❌ 极端减脂案例 (1个月瘦20斤)
❌ 不科学的产品推荐 (减肥药、瘦身贴)
❌ 身体羞辱内容
❌ 完美主义导向
❌ 比较与竞争导向
